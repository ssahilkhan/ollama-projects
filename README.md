# Ollama Projects

A collection of small projects built with Ollama and related technologies.

## Projects

### 1. Money Calculator
A simple expense/money calculator web app with a clean UI built using HTML, CSS, and JavaScript.

### 2. Chat App
A Node.js/Express chat application with MongoDB, user authentication (JWT), and real-time messaging capabilities.

## Tech Stack

- **Project 1**: HTML, CSS, JavaScript
- **Project 2**: Node.js, Express, MongoDB, JWT, Socket.io

## Features

- Expense/money calculation
- User authentication (register/login)
- Real-time messaging
- Persistent chat storage with MongoDB
- Middleware-based auth protection
- Clean, responsive UI
